
 - install npm
sudo apt install npm
 
 - install nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
 - then follow the instructions (have to run some commands)
 - update npm to v20
nvm install 20
nvm use 20

 - do initial build (ctrl + shift + b)

 - install vsce
npm install -g vsce

 - package the extension, do this in package root directory
   should produce some .vsix file
   make sure to bump the version in package.json before doing this
vsce package

 - login (should only have to ever do this once)
   this will request a Personal Access Token (PAT) which you can go to azure to generate
vsce login <publisher name>

 - publish
vsce publish
